• Designed and simulated a PWM signal generator in OrCAD PSpice using LT1363 op-amps, achieving an adjustable
duty cycle of 40%–80% and an output amplitude range of 3V–10V at 19kHz.
• Implemented a sawtooth wave generator using an integrator and non-inverting comparator, with asymmetric
charge/discharge via D1N4148 diodes to produce the reference waveform.
• Validated circuit robustness through Monte Carlo andWorst-Case analyses, confirming stable PWM output within
specifications across component tolerances of 10%.
